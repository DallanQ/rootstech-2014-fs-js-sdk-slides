Philosophy

Object model is probably obvious to FS experts, but wasn't obvious to me.
Understanding it has helped me immensely. I've added it to the SDK docs.
The main contribution of the SDK is it allows you to think of the FS REST API not as a set of REST calls that return JSON/XML, but as a set of function calls that return objects.
And because this is Javascript, the prototypes for those objects can be extended with whatever additional functions you want.

Promises

Are Awesome!

One of the biggest challenges for a java programmer working on the front-end is nearly everything in the browser is asynchronous: listening for user clicks, waiting for http responses
You have to learn to embrace asynchronicity. Promises make that much easier.
There are other ways of dealing with asynchronicity, but I think promises are the most elegant once you get to know them.
The SDK function calls return promises.

Think of a Promise as a Single-Use Event Bus


var promise = REST.asyncFunction();

promise.then(function(response) {
   // do something with the response
});
            

function asyncFunction() {
   var deferred = $.Deferred();

   setTimeout(function() {
      deferred.resolve("Response");
   }, 60*1000);

   return deferred.promise();
}

Handling Rejection


var promise = REST.asyncFunction();

promise.then(function(response) {
   // do something with the response
}, function(errorResponse) {
   // handle the error
});
            

function asyncFunction() {
   var deferred = $.Deferred();

   setTimeout(function() {
      deferred.reject("Error");
   }, 60*1000);

   return deferred.promise();
}

Another Example


function uploadFile(file) {
  var fd = new FormData();
  fd.append('artifact', file);
  // create memory
  return FamilySearch.createMemory(fd).then(function(memoryId) {

    // read memory to get memory artifact URL (memory.about)
    return FamilySearch.getMemory(memoryId).then(function(response) {
      var memory = response.getMemory();

      // create a memory persona from the person's name and the memory artifact URL
      var memoryPersona = new FamilySearch.MemoryPersona(person.$getDisplayName(), memory.about);

      // add the memory persona to the memory
      return memory.$addMemoryPersona(memoryPersona).then(function(memoryPersonaRef) {

        // add the memory persona reference to the person
        return person.$addMemoryPersonaRef(memoryPersonaRef).then(function() {
          // success!
          return memory;
        });
      });
    });
  });
};
